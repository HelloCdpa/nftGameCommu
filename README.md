NFT 게임 커뮤니티 프로젝트 기획안

작성일 :
2021. 12. 10

작성자 :
박혜린


1. 프로젝트 명 : NFT 게임 커뮤니티 프로젝트

2. 기획의도 

 최근 NFT시장의 성장으로 '디지털 자산'에 대한 사람들의 관심이 많아졌습니다. 

 기존의 게임시장에서는 게임의 인기가 아무리 많아져도 게임 회사만 돈을 벌고,

 게임유저들은 즐기기만 하고 오히려 과금을 하지만 얻는 자산은 아무것도 없었습니다.

 아무리 돈을 주고 아이템을 구매하여 내가 갖게 된다한들 게임 내의 아이템은 게임 회사가 망하면 다 사라지는 불안정한 

자산입니다. 이러한 독점적인 시장에서 NFT라는 디지털 자산으로 인해 이제는 게임 회사도 NFT거래 수수료로 돈을 벌고,

 유저들도 자신만이 소유할 수 있고 모든 곳에서 거래가 가능한 인증된 디지털 자산을 가질 수 있는 시장이 시작되었습니다. 

 공식적으로 한국에서는 이러한 NFT게임들을 인정하지 않고 있지만, 게임을 할 수는 있습니다. 

그러다보니 세계 시장과는 다르게 한국 유저가 적어서 게임 유저 간 정보 공유를 할 수 있는 커뮤니티가 많지 않은 것 같습니다. 

문제는 이렇게 국내에서 정보 공유가 원활하지 않다보니

1. 모든 것이 자리잡히지 않은 시장에 NFT게임이 중구난방으로 생기면서 사기를 당하는 사람들이 많이 생겼습니다.

2. 하고 싶은 NFT게임이 있어도 한국어 지원이 되지 않고, 시작하는 난이도가 높아서 다른 국가의 유저들에 비해 손해를 봅니다.

3. NFT 게임 정보를 한 군데에 모아 놓은 곳이 없어 게임의 소식을 전해 듣기 어렵습니다.

등등 이 외에도 많은 문제점이 있고 조금이나마 해결하려면 NFT게임만을 위한 커뮤니티 사이트를 만들어야 한다고 생각합니다.

기획할 프로그램은 NFT를 체험해보면서 NFT게임의 정보 공유를 할 수 있는 커뮤니티입니다.



3.벤치마킹

참고 할 사이트 - https://cobak.co.kr/ 코박 사이트의 커뮤니티



다른 커뮤니티보다 UI가 깔끔해서 보기가 좋습니다.

깔끔한 구성과 글쓰기/좋아요/ 댓글 기능을 참고하고 싶습니다.



오픈씨 (NFT그림 거래소)- https://opensea.io/collection/messari-2022-theses 

NFT 사진 프로필을 구매화면에 사용하고 싶은 구성과 구매 기능 (실제 구매는 포인트 차감)



4.     주요기능 

회원가입 - 이름 / 아이디 / 비밀번호 / 주로하는 게임 선택 / 이메일

로그인 - 이름 아이디

글쓰기 

글목록 - 최신글이면 제목 옆에 new 아이콘 붙이기

상세 글 조회 - 작성자의 프로필 사진과 함께 글 보여주기

댓글작성

조회수/좋아요 순으로 글 보기

글 / 댓글 작성하면 포인트 주기

포인트로 NFT그림/사진 프로필 구매

현금으로 포인트 충전(카카오 api를 통해)

마이페이지 - 프로필 사진 수정 / 회원 정보 업데이트 / 내가 쓴 글 보기

관리자 - 회원 삭제 / 게시글 삭제 권한 


5. 기타

프로필사진은 회원가입 시 기본 사진값으로 정해져 있고 포인트로 구매해야만 선택하여 등록할 수 있다.
